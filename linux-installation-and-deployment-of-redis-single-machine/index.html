Linux 安装、部署Redis（单机）

1.下载Redis

http://www.redis.cn/download.html 下载相应redis版本，并上传到服务器上

2.安装

解压，编译

1
2
3

tar -xzf redis-5.0.5.tar.gz
cd redis-5.0.5
make

编译完成后，通过如下命令开启Redis

1

src/redis-server

3.设置Redis远程连接，后台启动，设置密码

打开配置文件

1

vi redis.conf

设置后台启动

1

daemonize yes # no 修改为 yes

设置远程连接

1
2

bind 135.224.85.5 # 修改为主机ip
protected-mode no # yes 修改为 no

设置密码

1

requirepass 123456 # 设置密码为123456

4.重新开启Redis

关闭Redis命令

1
2

lsof -i :6379 # 找到对应的pid
kill -9 XXXX # 杀掉对应的pid进程

重启Redis命令

1

src/redis-server redis.conf # 进入redis目录下
